# 💰 Personal Finance Manager - Desktop App

## 📌 Description
This is a Java-based desktop application developed using Swing.  
It helps users manage their daily finances by recording income and expenses in a simple and interactive way.

The application also includes a forecasting feature that predicts next month's expenses using dynamic variation, making it more realistic.

---

## 🚀 Features
- Add Income  
- Add Expense with Category (Food, Travel, Shopping)  
- Show Financial Summary  
- View Transaction History  
- Expense Forecasting (Approximate Prediction)  
- User-friendly GUI  

---

## 🛠️ Technologies Used
- Java  
- Swing (GUI)  
- ArrayList  
- Event Handling  

---

## 📊 How It Works
- Users can add income and expenses  
- Expenses can be categorized  
- The system calculates total income, expense, and balance  
- Forecasting predicts future expenses using a dynamic formula
